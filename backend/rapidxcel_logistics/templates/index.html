RapidXcel Logistics API Documentation

Orders

Endpoint Method Description
/api/orders GET Retrieve a list of orders
/api/orders POST Create a new order
/api/orders/{id} GET Retrieve a specific order by ID
/api/orders/{id} PUT Update a specific order by ID
/api/orders/{id} DELETE Delete a specific order by ID
